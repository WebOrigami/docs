## Example: Defining a virtual folder

```console
$ ls
markdown    html = map(markdown, mdHtml, extension='md->html')
$ ls markdown
Alice.md    Bob.md    Carol.md
$ cat Alice.md
Hello, **Alice**.
```

```console
$ ori folder
html: !ori map(markdown, mdHtml, extension='md->html')
markdown:
  Alice.md: Hello, **Alice**.
  Bob.md: Hello, **Bob**.
  Carol.md: Hello, **Carol**.
```
